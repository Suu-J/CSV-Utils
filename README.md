# CSV Utility Scripts

Scripts for CSV file manipulation, comparison, and analysis.

## File Comparison
- **compareCSV.py** - Compares two CSV files to identify matching records
- **subtract_csv.py** - Subtracts one CSV dataset from another

## Data Extraction
- **fetch_file_names.py** - Extracts file names from directories to CSV
- **rows_count.py** - Counts rows in text files

## Dependencies
- pandas (for most scripts)
- csv (Python standard library)

## Use Cases
- Data validation and verification
- Finding differences between datasets
- Extracting specific information from large files
- Preparing data for processing
